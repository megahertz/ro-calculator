Formulas for verification
dose_mL = drops / dropsPerMl dilution_factor = bottle_mL / dose_mL C_stock(mg/L) = C_target(mg/L) * dilution_factor grams_salt_per_L_stock = (C_stock(mg/L) / 1000) / mass_fraction_ion_in_salt grams_in_stock_volume = grams_salt_per_L_stock * (stock_mL / 1000) Hardness: Ca as CaCO₃ (mg/L) = Ca(mg/L) * 2.497 Mg as CaCO₃ (mg/L) = Mg(mg/L) * 4.118 GH(°dH) = (CaCO3_total mg/L) / 17.848 Alkalinity (approx): HCO₃ as CaCO₃ (mg/L) = HCO₃(mg/L) * (50 / 61.0168) KH(°dH) = (HCO3_as_CaCO3 mg/L) / 17.848 Approx added TDS: TDS_added ≈ (mg/L of added salts in finished water summed)
